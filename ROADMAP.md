# ROADMAP — 클립샷(ClipShot)

## 앱의 목적

화면 캡처를 "저장 = 클립보드"로 재정의한다. 캡처하면 즉시 클립보드에 들어가 어디든 바로
붙여넣을 수 있고, 동시에 최근 30개가 자동 보관되어 지난 캡처를 언제든 다시 꺼내 쓸 수 있는,
가볍고 직관적인 Windows·macOS 무료 유틸리티.

## 핵심 기능

- 4가지 캡처 모드: 영역 / 창 / 디스플레이 / 모든 디스플레이
- 캡처 즉시 클립보드 복사 + 최근 N개(기본 30) 자동 히스토리(링버퍼)
- 히스토리 뷰어: 다시 복사 / 파일 내보내기(PNG·JPG) / 삭제 / 전체 비우기
- 전역 단축키(재바인딩) + 트레이 상주
- 효과/옵션: 플래시, 셔터 소리, 완료 토스트, 시작 프로그램 등록, 한/영

## 디자인 원칙

- 가볍게: Tauri로 작은 바이너리·낮은 메모리. 불필요한 의존성 배제.
- 즉시성: 캡처 → 붙여넣기까지 마찰 0. 별도 저장 단계 강요하지 않음.
- 로컬 전용·프라이버시: 외부 전송·추적 없음. 처리는 전부 기기 내.
- 단일 코드베이스로 Windows/macOS 동시 지원.
